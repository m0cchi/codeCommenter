Code Commenter

Code Commenterとは

preタグにコメントをすることができるようになります.

導入方法は簡単!
以下のような表示ができます.

$(document).ready(function(){ $('#code').comment({ click :function (prePointer,commentText){ prePointer.next = $('<p/>').text(commentText); document.getElementById('code').insertBefore( prePointer.next.get()[0], prePointer.current.get()[0].nextSibling ); } }); });
